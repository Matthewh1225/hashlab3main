Hash Function Comparison:
BurrisHash:
  - Severe clustering (especially at slot 33)
  - Poor distribution characteristics
  - Linear: 21.20 avg probes
  - Random: 4.08-4.37 avg probes
  
YourHash:
  - Excellent distribution
  - Minimal clustering
  - Linear: 1.71 avg probes (91.9% improvement)
  - Random: 2.05-2.35 avg probes (42.5-53.0% improvement)

Probe Method Comparison:
Linear Probing:
  - Creates long probe chains with poor hash functions
  - Magnifies clustering problems
  - Excellent with good hash functions (YourHash)
  
Random Probing:
  - Significantly reduces clustering impact
  - More consistent performance
  - Less predictable but generally better with poor hash functions

Key Insights:
1. Hash function quality is the dominant factor
2. Good hash function + linear probing = excellent performance
3. Poor hash function + linear probing = catastrophic performance
4. Random probing provides safety net for poor hash functions
5. YourHash achieves near-perfect distribution (1.16 first 25, 2.40 last 25)
6. BurrisHash suffers from systematic bias toward certain slots
